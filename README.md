# 🎮 Tic Tac Toe (Python)

A simple terminal-based Tic Tac Toe game built while learning Python. This project was one of my first steps into programming and helped me understand how to structure a complete Python application using functions and control flow.

---

## 📌 Features

- Two-player gameplay
- Randomly selects which player goes first
- Player chooses X or O
- Win detection across rows, columns, and diagonals
- Detects draw games
- Replay option after each match
- Input validation for board positions

---

## 🛠️ Technologies Used

- Python 3
- `random` module

---

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Tic-Tac-Toe.git
```

2. Navigate into the project folder

```bash
cd Tic-Tac-Toe
```

3. Run the program

```bash
python TIC_TAC_TOE.py
```
